Intro :📝

This api is used to display animal images of different sizes and you can also specify these sizes

/{name}/{width}/{height}

examples :📝

Cat : 🐱

In order to get pictures of cats of different sizes, you must put the word "cat" first, then write the required width, and then write the height

/cat/500/300

Dog :🐶

In order to get pictures of dogs of different sizes, you must put the word "dog" first, then write the required width, and then write the height

/dog/500/300

Deer :🦌

In order to get pictures of deers of different sizes, you must put the word "deer" first, then write the required width, and then write the height

/deer/500/300
